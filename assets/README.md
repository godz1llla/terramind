# Assets Directory

## Структура

```
assets/
├── images/          # Изображения приложения
│   └── (здесь будут фото растений, логотипы и т.д.)
├── icons/           # Иконки приложения
│   └── (здесь будут кастомные иконки)
```

## Для хакатона

В данный момент приложение использует:
- Material Icons (встроенные иконки Flutter)
- Placeholder'ы для изображений

## Добавление изображений

Чтобы добавить изображения:

1. Поместите файлы в соответствующую папку
2. Используйте в коде:
```dart
Image.asset('assets/images/your_image.png')
```

## Рекомендуемые размеры

- **Icons:** 24x24, 48x48, 96x96 px
- **Images:** 1080x1920 px (портрет), 1920x1080 px (ландшафт)
- **Thumbnails:** 300x300 px

## Форматы

- PNG для иконок и изображений с прозрачностью
- JPEG для фотографий
- SVG для векторных иконок (требует flutter_svg)
